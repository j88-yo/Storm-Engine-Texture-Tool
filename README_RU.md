# Storm Engine Texture Tool

Утилита для просмотра и конвертации текстур с типами файлов `.tga` и `.tga.tx`.

Предназначена для работы с текстурами при создании пользовательских модификаций, тестовых материалов и некоммерческих сборок игр на базе движка Storm Engine (https://github.com/storm-devs/storm-engine)

Программа позволяет:
- просматривать текстуры и сведения о них, включая разрешение, формат хранения, глубину цвета, тип alpha-канала, количество mip-уровней и используемый метод сжатия;
- конвертировать текстуры между TGA и TX в разрешении до 16K, включая создание mip-уровней;
- использовать сжатие DXT1 / DXT3 / DXT5;
- выполнять массовую конвертацию файлов;
- использовать программу для открытия `.tga.tx` напрямую из проводника Windows.

Для сжатия текстур используется актуальная версия NVIDIA Texture Tools (NVTT 3.2.5, 2024).

# Поддерживаемые форматы хранения

## A8R8G8B8

Несжатый 32-битный формат.

- максимальное качество изображения без потерь;
- прозрачность 8-битный alpha-канал;
- самый большой размер файла.

Примерный размер текстуры 1024×1024 — ~4 МБ.

---

## X8R8G8B8

Несжатый 32-битный формат.

- прозрачность отсутствует;
- максимальное качество изображения без потерь;
- самый большой размер файла.

Примерный размер текстуры 1024×1024 — ~4 МБ.

---

## A4R4G4B4

16-битный формат.

- прозрачность 4-битный alpha-канал с ограниченной плавностью;
- сниженная точность цвета по сравнению с 32-битными форматами;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## A1R5G5B5

16-битный формат.

- прозрачность 1-битный alpha-канал;
- прозрачность работает только в режимах «видимый/невидимый» пиксель;
- сниженная точность цвета;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## R5G6B5

16-битный формат.

- прозрачность отсутствует;
- сниженная точность цвета по сравнению с 32-битными форматами;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## DXT1

Сжатый формат.

- 4-битное блочное сжатие;
- прозрачность 1-битный alpha-канал;
- прозрачность работает только в режимах «видимый/невидимый» пиксель;
- заметная потеря качества изображения;
- минимальный размер файла.

Примерный размер текстуры 1024×1024 — ~512 КБ.

---

## DXT3

Сжатый формат.

- 8-битное блочное сжатие;
- прозрачность 4-битный alpha-канал;
- качество выше по сравнению с DXT1;
- возможны артефакты на плавных переходах прозрачности.

Примерный размер текстуры 1024×1024 — ~1 МБ.

---

## DXT5

Сжатый формат.

- 8-битное блочное сжатие;
- прозрачность 8-битный alpha-канал;
- оптимальное соотношение качества и размера;
- наиболее универсальный формат.

Примерный размер текстуры 1024×1024 — ~1 МБ.

DXT5 является рекомендуемым форматом для большинства текстур.

# Работа с программой

## Просмотр текстур

Для просмотра текстуры выберите файл `.tga` в левой или `.tga.tx` в правой файловой панели.

Изображение автоматически отобразится в области preview.

Поддерживается:
- масштабирование (колесом мыши, ползунком, кнопками в области preview, 1:1 / 100% масштаб, вписать в область preview);
- листание текстур (кнопками интерфейса и клавишами влево / вправо);
- перетаскивание изображения;
- смена фона (светлый, тёмный, шахматный);
- режим просмотра во весь экран (скрытие панелей интерфейса кнопкой полноэкранного режима или кнопками скрытия панелей, возврат стандартного режима по клавише `ESC`).

---

## Работа с папками и файлами

Левая файловая панель предназначена для работы с файлами `.tga`, правая — для `.tga.tx`.

Поддерживается:
- навигация по папкам кнопками (назад / вперёд, вверх, «Мой компьютер», «Загрузки», диски), дополнительными боковыми кнопками мыши (действия назад / вперёд);
- закрепление и открепление папок (через контекстное меню или иконку слева от папки);
- раздел с закреплёнными папками (кнопка `★`);
- массовая конвертация выделенных файлов;
- работа с файлами через контекстное меню:
  - открыть папку файла;
  - открыть папку в проводнике Windows;
  - переименовать;
  - удалить.

# Конвертация

## Конвертирование в `.tga.tx`

Для конвертирования в `.tga.tx` выберите `.tga` файл в левой файловой панели и нажмите кнопку «Собрать TX».

Поддерживается:
- выбор формата хранения;
- использование сжатия DXT1 / DXT3 / DXT5;
- создание mip-уровней;
- замена существующих файлов;
- создание копий при совпадении имён.

Конвертирование в `.tga.tx` поддерживается для текстур с размерами, соответствующими степени двойки.
Например: 256×256, 512×1024, 2048×2048, 4096×8192, 16384×16384.

---

## Конвертирование в `.tga`

Для конвертирования в `.tga` выберите `.tga.tx` файл в правой файловой панели и нажмите кнопку «Распаковать TGA».
Конвертирование выполняется в несжатый 32-битный формат с alpha-каналом A8R8G8B8 с сохранением оригинального качества изображения без потерь.

---

## Mip-уровни

Поддерживаются режимы:
- только базовый уровень (1 mip-уровень);
- полная mip-цепочка;
- пользовательское количество mip-уровней.

Mip-уровни используются для корректного рендера текстур при изменении расстояния до объекта. Их количество зависит от размера текстуры.
Рекомендуется использовать полную mip-цепочку для большинства текстур.

---

## Массовая конвертация

Для массовой конвертации выделите несколько файлов в файловой панели и нажмите кнопку «Собрать TX» или «Распаковать TGA».

Поддерживается:
- одновременное конвертирование нескольких файлов;
- замена существующих;
- создание копий;
- пропуск конфликтующих файлов.

Во время конвертации отображается процесс обработки и количество обработанных текстур.
После завершения операции отображается итоговый результат обработки.
