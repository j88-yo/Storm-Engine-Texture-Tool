# Storm Engine Texture Tool

Утилита для просмотра и конвертации текстур с типами файлов `.tga`, `.png` и `.tga.tx`.

Предназначена для работы с текстурами при создании пользовательских модификаций, тестовых материалов и некоммерческих сборок игр на базе движка Storm Engine ([GitHub Storm Engine](https://github.com/storm-devs/storm-engine)).

Программа позволяет:
- просматривать текстуры и сведения о них, включая разрешение, формат хранения, глубину цвета, тип alpha-канала, количество mip-уровней и используемый метод сжатия;
- конвертировать текстуры `.tga` и `.png` в `.tga.tx` в разрешении до 16K, включая создание mip-уровней;
- распаковывать `.tga.tx` в `.tga` или `.png` без потерь качества;
- использовать сжатие DXT1 / DXT3 / DXT5;
- выполнять одиночную и массовую конвертацию файлов;
- использовать удобный массовый сценарий апскейла текстур: распаковка `.tga.tx` в `.png`, обработка во внешнем upscaler и обратная сборка в `.tga.tx`;
- открывать файлы `.tga` и `.tga.tx` напрямую из проводника Windows.

Для сжатия текстур используется актуальная версия NVIDIA Texture Tools (NVTT 3.2.5, 2024).

# Поддерживаемые форматы хранения

## A8R8G8B8

Несжатый 32-битный формат.

- максимальное качество изображения без потерь;
- прозрачность: 8-bit alpha;
- самый большой размер файла.

Примерный размер текстуры 1024×1024 — ~4 МБ.

---

## X8R8G8B8

Несжатый 32-битный формат.

- no alpha;
- максимальное качество изображения без потерь;
- самый большой размер файла.

Примерный размер текстуры 1024×1024 — ~4 МБ.

---

## A4R4G4B4

16-битный формат.

- прозрачность: 4-bit alpha с ограниченной плавностью;
- сниженная точность цвета по сравнению с 32-битными форматами;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## A1R5G5B5

16-битный формат.

- прозрачность: 1-bit alpha;
- прозрачность работает только в режимах «видимый / невидимый» пиксель;
- сниженная точность цвета;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## R5G6B5

16-битный формат.

- no alpha;
- цвета менее насыщенные по сравнению с 32-битными форматами;
- компактный размер файла.

Примерный размер текстуры 1024×1024 — ~2 МБ.

---

## DXT1

Сжатый формат.

- сжатие, no alpha;
- минимальный размер файла;
- возможны заметные артефакты.

Примерный размер текстуры 1024×1024 — ~512 КБ.

---

## DXT3

Сжатый формат.

- сжатие, резкая alpha;
- подходит для текстур с чёткими границами alpha-канала;
- примерный размер текстуры 1024×1024 — ~1 МБ.

---

## DXT5

Сжатый формат.

- сжатие, плавная alpha;
- без заметных потерь качества;
- рекомендуется по умолчанию;
- примерный размер текстуры 1024×1024 — ~1 МБ.

# Работа с приложением

## Preview текстур

Чтобы открыть preview текстуры, выберите `.tga` или `.png` в левой файловой панели либо `.tga.tx` в правой панели.

Поддерживается:
- масштабирование колесом мыши, слайдером и кнопками `1:1`, `100%`, `Вписать`;
- листание текстур кнопками интерфейса и клавишами `←`, `→`, `↑`, `↓`;
- перетаскивание изображения мышью;
- переключение фоновых режимов preview;
- полноэкранный режим просмотра с выходом по `ESC`.

---

## Работа с файлами и папками

Левая файловая панель предназначена для `.tga` и `.png`, правая — для `.tga.tx`.

Поддерживается:
- навигация по папкам кнопками `Назад`, `Вперёд`, `Вверх`, `Мой компьютер`, `Загрузки`, а также боковыми кнопками мыши;
- навигация по спискам файлов клавишами `↑`, `↓`, `←`, `→`;
- сортировка файлов по имени и по типу;
- закрепление и открепление папок через контекстное меню или иконку слева от папки;
- раздел с закреплёнными папками (`★`);
- перетаскивание мышью для изменения порядка закреплённых папок;
- открытие окна конвертации клавишей `Enter`, подтверждение действий клавишей `Enter`, удаление выбранных файлов клавишей `Delete`;
- действия через контекстное меню: открыть папку файла, открыть папку в проводнике Windows, переименовать, удалить.

# Конвертация

## Конвертирование в `.tga.tx`

Для конвертирования в `.tga.tx` выберите файл `.tga` или `.png` в левой файловой панели и нажмите кнопку «Собрать TX».

Поддерживается:
- выбор формата хранения;
- сжатие DXT1 / DXT3 / DXT5;
- создание mip-уровней;
- замена существующих файлов;
- создание копий при совпадении имён;
- расчёт общего размера выбранных файлов до и после конвертации.

Конвертирование в `.tga.tx` поддерживается для текстур со сторонами, соответствующими степеням двойки.
Например: 256×256, 512×1024, 2048×2048, 4096×8192, 16384×16384.

---

## Распаковка `.tga.tx`

Для распаковки `.tga.tx` выберите нужный файл в правой файловой панели и нажмите кнопку «Распаковать».

В окне распаковки можно выбрать выходной формат:
- `.tga` — TGA 32-bit, 8-bit alpha, без потерь качества;
- `.png` — PNG 32-bit, 8-bit alpha, без потерь качества.

---

## Mip-уровни

Поддерживаются режимы:
- только базовый уровень (1 mip-уровень);
- полная mip-цепочка;
- пользовательское количество mip-уровней.

Mip-уровни используются для корректного рендера текстур при изменении расстояния до объекта. Их количество зависит от размера текстуры.
Для большинства текстур рекомендуется использовать полную mip-цепочку.

---

## Массовая конвертация

Для массовой конвертации выделите несколько файлов в файловой панели и нажмите кнопку «Собрать TX» или «Распаковать».

Поддерживается:
- одновременное конвертирование нескольких `.png` и `.tga` файлов;
- замена существующих файлов;
- создание копий;
- пропуск конфликтующих файлов.

Во время конвертации отображается прогресс обработки и количество обработанных текстур.
После завершения операции отображается итоговый результат обработки.
