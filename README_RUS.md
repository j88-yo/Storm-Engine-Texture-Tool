# Storm Engine Texture Tool 0.5.2 — Руководство пользователя

# 1. Введение

Storm Engine Texture Tool — утилита для просмотра и конвертации текстур между форматами `.tga.tx` и `.tga`.

Программа позволяет:
- просматривать текстуры;
- просматривать параметры текстуры;
- конвертировать TGA ↔ TX;
- конвертировать TX с mip-уровнями;
- использовать сжатие DXT1 / DXT3 / DXT5;
- выполнять массовую конвертацию файлов.

Утилита использует двухпанельный интерфейс со встроенной областью preview и поддерживает основные режимы сжатия, используемые в движке Storm Engine.

Для сжатия текстур используется библиотека NVIDIA Texture Tools.

Поддерживаемые форматы:
- `.tga`
- `.tga.tx`

---

# 2. Работа с программой

## Просмотр текстур

Выберите файл `.tga` или `.tx` в одной из файловых панелей.

Изображение автоматически отобразится в области preview.

Поддерживается:
- масштабирование колесом мыши;
- сброс масштаба;
- режим 1:1;
- режим «Вписать»;
- перетаскивание изображения;
- отображение alpha-канала;
- скрытие панелей интерфейса.

Если скрыта хотя бы одна панель интерфейса, клавиша `ESC` возвращает стандартный режим отображения.

---

## Файловые панели

Файловые панели поддерживают:
- навигацию по папкам;
- закреплённые папки;
- множественное выделение;
- кнопки мыши «Назад» и «Вперёд»;
- контекстное меню:
  - открыть;
  - открыть папку файла;
  - переименовать;
  - удалить.

---

## Информационный блок

Информационный блок отображает:
- разрешение текстуры;
- формат;
- тип alpha;
- bit depth;
- количество mip-уровней;
- размер файлов.

Для части параметров доступны всплывающие подсказки с описанием.

---

# 3. Конвертация

## Сборка TX

Для сборки `.tga.tx` доступны форматы:

Без сжатия:
- A8R8G8B8
- X8R8G8B8
- A4R4G4B4
- A1R5G5B5
- R5G6B5

Сжатие DXT:
- DXT1
- DXT3
- DXT5

Для сжатия используется библиотека NVIDIA Texture Tools.

---

## Mip-уровни

Поддерживаются режимы:
- только базовый уровень;
- полная mip-цепочка;
- пользовательское количество mip-уровней.

Mip-уровни используются для более корректного отображения текстур на дальних дистанциях.

Для генерации mip-уровней текстура должна иметь размеры со сторонами степени двойки.

---

## Массовая конвертация

Поддерживается одновременная обработка нескольких файлов.

Во время конвертации доступны:
- замена существующих файлов;
- создание копий;
- пропуск конфликтующих файлов.

После завершения операции отображается итоговый результат обработки.

---

# 4. Дополнительно

Приложение может использоваться как viewer по умолчанию для файлов `.tga.tx`.

Назначение программы по умолчанию доступно в разделе «О программе».

---

# 5. Ограничения

Утилита поддерживает работу с текстурами, размеры которых соответствуют степени двойки.

Некоторые текстуры могут использовать нестандартные параметры хранения.

DXT-сжатие может влиять на качество изображения.
